# 프로젝트 기준 스택

현재 `package.json` 기준 프로젝트 스택은 아래와 같다.

- Vue 3
- TypeScript
- Vue Router 4
- Pinia
- `@tanstack/vue-query`
- Tailwind CSS
- ESLint
- Prettier
- Vite
- moment
- moment-timezone

---

## 메모

- 문서에서 라우팅 예시는 모두 `vue-router` 기준으로 작성한다.
- 서버 상태 예시는 `@tanstack/vue-query` 기준으로 작성한다.
- 현재 코드베이스에는 D3, AG Grid, Data Table 의존성이 없다. 관련 문서는 도입 시 확장 기준으로만 본다.
